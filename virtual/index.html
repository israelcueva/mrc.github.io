Horarios

Para poder ingresar a la plataforma debes tomar en cuenta el horario de restricción de ingreso, si no puedes ingresar, verifica que según el horario de restricción tu aula se encuentre habilitada. El resto de horarios es para indicarte la disponibilidad de los profesores para tus consultas y que estas se resuelvan de inmediato.

Si deseas ver los horarios con mayor detenimiento, revisa esta página.

Acceso

PRIMERO: Credenciales de acceso

Para que puedas tener acceso a la plataforma debemos verificar, primero, que seas el apoderado del alumno del cual se desea el acceso.

Para ello se ha dipuesto que cada tutor, valide la información del padre, y así poder entregarle el acceso.
